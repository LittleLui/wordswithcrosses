Robo Crosswords v2.5.3

Hi all. Thanks to everyone who wrote in about the Joseph, Sheffer, and Premier puzzles failing to download (as well as the NYT from way back when). The publisher of those puzzles has moved them to a new download location.

The NYT added a CAPTCHA to their login page a while back, so the app is no longer able to automatically log in. Support for that is still a work in progress, and when finished will require you to log in via an in-app web view. Thanks for your continued patience on that.

This release raised the minimum supported Android version from 1.6 (Donut) to 2.3 (Gingerbread). I'll likely raise it to 4.0.3 (Ice Cream Sandwich) in the future, which is currently the oldest version that has any active installs. This release also raised the target SDK version from 4.2 (Jelly Bean) to 8.0 (Oreo), as required by Google Play, which may have caused some unintended behavior changes, so please be on the lookout for anything unexpected.

As always, if you enjoy using Robo Crosswords, please consider rating it in the Play Store.

Robo Crosswords is a free and open-source project. You can obtain the source code, submit issues, and submit pull requests from the project page on GitHub. You can also email me at <robocrosswords@adamrosenfield.com> if you'd like to provide any feedback, bug reports, feature requests, or new puzzle sources, etc.

Keep circulating the tapes.

Welcome to Robo Crosswords! The wonderful Shortyz Crosswords app has not seen any updates in quite some time, so took it upon myself to fork the project into what you see before you. That that was possible at all is thanks to Shortyz' being open-source.

Many thanks to Robert Cooper for creating Shortyz.

Robo Crosswords is free, will always remain free, and contains no advertising. I don't want to give the appearance that I'm reselling anyone else's property. I don't host or copy the puzzles you download in the app in any way—you download them directly from their publicly available sources with the app. This is a side project of mine done in my spare time.

That said, if you'd like to make a donation to help further the development, please tap the "Donate" link above. If you'd prefer to donate from a different computer or device, you can also donate at http://adamrosenfield.com/donate. Also consider purchasing a subscription to one of the many great paid puzzle sources listed here.
